# NexusMedia Website

Marketing agency website built with Vite, React, TypeScript, and Tailwind CSS.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui components

## Getting Started

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Test

```sh
npm run test
```
